# Northwind-Proyecto-final-de-Power-Bi-
## Descripción
Proyecto escolar realizado en Power Bi sobre las ventas y pedidos de la empresa Northwind, haciendo enfasis en el análisis de las cifras de ventas por País y ciudad, productos mas vendidos, comportamiento de las ventas a lo largo del tiempo, etc.

## Proceso
Se obtuvo la base de datos (1996 - 1998) sobre las ventas de la empresa Northwind, se cargó en Power Bi y los datos fueron depurados eliminando variables no  relevantes para el objetivo del proyecto.

Se crearon nuevas variables con resultados de operaciones aritméticas utilizando DAX y se crearon distintos gráficos dinámicos (De pastel, de barras, de lineas, mapa de calor, etc) para facilitar la visualización y comprensión de los datos.

## Páginas
El dashboard se compone de 2 páginas principales:

1.- Vista general (Todos los gráficos dinámicos sobre las ventas de la empresa).

2.- Detalles por cliente (Tabla con información de los pedidos agrupados por cliente, monto, etc).

## Herramientas utilizadas.
*Power Bi desktop.

*Base de datos proporcionada.
